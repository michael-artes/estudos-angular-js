O capitulo ensinou sobre:

ng-disable e expressão: desabilida elementos html
ng-options e expressão: trabalha com options de selects (as e group by)
ng-class: aplica classe em elementos html
ng-style: aplica um estilo css no elemento dinamicamente

ng-show: se condicao for verdadeira mostrará o elemento
ng-hide: se condicao for verdadeira ocultará o elemento
ng-if: funciona semelhante ao <c:if> display tag, o elemento só irá existir
se a condição for verdadeira

ng-include: faz o include de uma página!!!
